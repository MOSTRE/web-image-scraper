# Web Image Scraper

Web Image Scraper is a Python script that allows you to scrape images from a given website URL and save them to a specified directory.

## Features

- Scrapes images from a webpage using BeautifulSoup.
- Downloads images and saves them to a directory.
- Easy to use with a simple command-line interface.

## Requirements

- Python 3.x
- requests library (`pip install requests`)
- BeautifulSoup library (`pip install beautifulsoup4`)
